- 같은 수 같은 자리 = 스트라이크
- 같은 수 다른 자리 = 볼
- 같은 수 아예 없음 = 포볼
- 컴퓨터는 임의의 3개 숫자 선택
- 사용자 반복해 3개 숫자 모두 맞히면 (=3스트라이크)게임 종료
